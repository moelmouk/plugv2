# Icons pour Macro Recorder Pro

Ce dossier doit contenir les icônes suivantes pour l'extension Chrome:

## Icônes requises

- `icon16.png` - 16x16 pixels
- `icon32.png` - 32x32 pixels  
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

## Icônes optionnelles (pour les états)

- `icon-recording16.png`, `icon-recording32.png`, etc. (point rouge pour l'enregistrement)
- `icon-playing16.png`, `icon-playing32.png`, etc. (point vert pour la lecture)
- `icon-paused16.png`, `icon-paused32.png`, etc. (point orange pour la pause)

## Design suggéré

L'icône principale devrait être:
- Un cercle représentant l'enregistrement (comme un bouton record)
- Couleur principale: bleu (#3b82f6) vers violet (#8b5cf6)
- Style moderne et minimaliste

## Créer les icônes

Vous pouvez utiliser un outil comme:
- Figma
- Adobe Illustrator
- Canva
- GIMP

Ou convertir un SVG en PNG avec différentes tailles.
